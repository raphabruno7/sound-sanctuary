# Media Slots

## Folder Purpose
- `public/media/hero`: assets intended for top-of-page hero contexts.
- `public/media/sections`: assets intended for supporting content sections.
- `public/media/portfolio`: assets intended as proof/context in portfolio content.

## Image Roles
- Hero image role: primary contextual introduction for a route.
- Section image role: supplemental context for a specific module/section.
- Proof image role: evidence-oriented media supporting trust claims.

## Implementation Note
- This folder structure defines content/media slots only.
- Visual styling, treatments, and rendering behavior will be defined in later design-focused work.
