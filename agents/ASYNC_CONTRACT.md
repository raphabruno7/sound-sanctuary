# Async Agent Contract

## How agents work in this repo
- Pick ONE task only
- Create ONE branch per task
- Follow the corresponding runbook
- Do NOT touch unrelated files
- Open PR when done

## Naming
Branch: feat/<task-id>

Example:
feat/S01-0001-hero-photography

## Required before PR
- Run scripts/agent_check.sh
- Ensure acceptance criteria met
- Add short PR description

## What agents must NOT do
- No refactors outside scope
- No aesthetic drift
- No dependency bloat
